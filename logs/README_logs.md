# Log Notes

This folder explains how diagnostic evidence is handled in this portfolio.

## Raw logs
- Original CAN logs exist.
- Raw logs are **not published** in this repository.

## Public vs private
- Public:
  - Report PDFs
  - Sanitized log excerpts
- Private:
  - Full raw CAN logs
  - Any data that could identify vehicle, ECU, or OEM

## Sanitization
Before publishing, the following were removed or generalized:
- VIN / ECU identifiers
- OEM-specific values
- Timestamps not required for analysis

## Access
Raw logs can be reviewed **upon reasonable request** for technical discussion or verification.

# 증거 자료 안내

이 폴더는 본 포트폴리오에서 사용된 진단 증거 자료의 처리 방식을 간단히 설명합니다.

## 원본 로그
- 원본 CAN 로그는 존재합니다.
- 원본 로그는 이 저장소에 공개하지 않습니다.

## 공개 / 비공개 기준
- 공개:
  - 분석 리포트 PDF
  - 식별 정보가 제거된 로그 발췌본
- 비공개:
  - 전체 원본 CAN 로그
  - 차량, ECU, OEM을 식별할 수 있는 데이터

## 익명화
공개 전 다음 항목을 제거 또는 일반화했습니다.
- VIN, ECU 식별자
- OEM 고유 값
- 분석에 불필요한 타임스탬프

## 접근
기술 검토 목적의 합리적인 요청이 있을 경우,
원본 로그는 별도로 검토 가능합니다.
