# Automotive UDS (Extended CAN) Failure Analysis Portfolio

**Description:**  
This repository documents a log-based reverse engineering project focused on automotive diagnostic communication failures using UDS over Extended CAN.

**Key Demonstrated Skills**

- ISO 14229 (UDS) protocol analysis
    
- OEM-specific behavior identification
    
- Failure pattern classification
    
- Technical documentation and traceability

**Confidentiality**

- Raw logs are anonymized and sanitized
    
- Proprietary identifiers are excluded
    
- Cryptographic details are intentionally omitted


# 자동차 UDS(Extended CAN) 진단 통신 실패 분석 포트폴리오

**개요:**  
본 저장소는 UDS 기반 자동차 진단 통신 실패 사례를  
통신 로그만을 활용하여 분석한 리버스 엔지니어링 프로젝트를 담고 있습니다.

**핵심 역량**

- ISO 14229(UDS) 프로토콜 분석
    
- OEM 예외 로직 식별
    
- 통신 실패 패턴 분석
    
- 기술 문서화 및 추적성 확보

**보안 및 기밀성**

- 원본 로그는 비공개
    
- 모든 식별 정보 비식별화
    
- 보안 알고리즘 세부 내용 미포함
